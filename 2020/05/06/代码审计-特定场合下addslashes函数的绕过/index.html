addslashes 与escapeshellcmd函数

这2个函数都是在需要转义的字符前加上反斜杠注释，具体瞅瞅：

addslashes — 使用反斜线引用字符串

string addslashes ( string $str )

作用：在单引号（’）、双引号（”）、反斜线（\）与 NUL（ NULL 字符）字符之前加上反斜线。

而escapeshellcmd它主要是给shell提供转义：

反斜杠会在以下字符之前插入： #&;`|?~<>^()[]{}$, x0A 和 xFF。 ‘ 和 “ 仅在不配对儿的时候被转义。 在 Windows 平台上，所有这些字符以及 % 都会被空格代替。（译注：实际测试发现在 Windows 平台是前缀 ^ 来转义的。）

substr 函数

substr — 返回字符串的子串

string substr ( string $string , int $start [, int $length ] )

作用：返回字符串 string 由 start 和 length 参数指定的子字符串。

基础知识

  • 对于传入的非法的 $_GET 数组参数名，PHP会将他们替换成 下划线。
  • 当我们使用HPP（HTTP参数污染）传入多个相同参数给服务器时，PHP只会接收到后者的值。（这一特性和中间件有关系）
  • 通过 $_SERVER[‘REQUEST_URI’] 方式获得的参数，并不会对参数中的某些特殊字符进行替换。

读题

WAF1 是对$_REQUEST传进来的value进行过滤，检测到非法字符直接退出。

WAF2 是通过$_SERVER['REQUEST_URI'] 的方式获取参数，然后使用 explode 函数针对 & 进行分割，获取到每个参数的参数名和参数值。然后针对每个参数值调用 dhtmlspecialchars() 函数进行过滤。
