漏洞分析

分析代码，可以发现我们传入的数据会被实例化为xml,那我们肯定想到 xxe 。但是题目中strpos过滤了<、>符号，我们怎么办呢？
我们看看 strpos函数的定义：

strpos — 查找字符串首次出现的位置
作用：主要是用来查找字符在字符串中首次出现的位置。
结构：int strpos ( string $haystack , mixed $needle [, int $offset = 0 ] )
值的注意的是第一位是0，也就是说如果要查找的字符在第一位就会返回 0

通过上面的定义我们知道，如果我们把< >第一位就出现，我们就可以绕过了。
