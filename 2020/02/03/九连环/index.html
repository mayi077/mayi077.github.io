九连环

发表于 2020-02-03 更新于 2020-02-15 分类于 CTF复现 阅读次数：
本文字数： 426 阅读时长 ≈ 1 分钟

前言：

 通过这题我们可以学习一款新工具，steghide，他可以帮我们提取出一些隐藏文件（大概是用他这个软件隐藏的，不限于此），接下来我们来看看这题吧。

解题思路

  1. 下载文件，得到一个图片开始以为文件名是什么的密码，结果然并luan。我们用foremost分解出一个压缩包，里面有一个图片和一个压缩包，压缩包没加密，图片加密了。
  2. 丢进winhex对比zip构造图，发现压缩方式处为0000 也就是说没有加密方式那么就是伪加密。我们按照以前的知识可以删除伪加密，解出压缩包。
  3. 因为qwe.zip有密码，而又只给了我们一个good-宸插悎骞图片所以从图片下手。经过多种方法无果。
  4. 通过wp得知可以用 steghide 来提取出一个文件。kali安装steghide后使用
    steghhide extract -sf "文件名"
    回车后提示要密码，我们这里没设置密码直接回车即可。得到ko.txt。文件内容为：
  5. 然后去解压qwe.zip，得到flag.txt
