shlle中的if和sleep

此时我们发送一个 cmd=sleep 3 去再次确认我们的命令是否可以被执行，得到的结果是被执行了只是没有回显而已，但是反弹shell又用不了，怎么办呢？
我们想到了 if 语句，例如 if [ 1 = 1 ];then sleep 3;fi 这里注意一下if的格式，空格是必须的。

我们可以利用这种思路盲注得到我们命令执行的结果，
但是我们怎么去在if中执行命令且指定其所取位置？
