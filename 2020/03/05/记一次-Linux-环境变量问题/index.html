问题描述

 使用 pip3 安装扩展后无法直接使用。比如我们使用pip install ipython 安装后（使用 pip list 查看 ipython存在），应该可以直接在任何位置使用 ipython 这个命令了。但是今天我用 pip3 install 安装的却无法直接使用，报错命令不存在。现在知道了，这一般都是 $PATH 的问题啦。（前题是确实安装成功了）

Linux 中的 $PATH 是啥？

 $PATH环境变量是以冒号分隔的目录列表，用于告诉shell搜索可执行文件的目录。和 windows 下的环境变量 path 一个概念，就是你使用这个命令时它到你给定的位置去找可执行文件有多个会优先用前面的，如果都没找到就报错命令不存在。
 一般我们使用 echo $PATH来查看环境变量。
