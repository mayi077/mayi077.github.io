[BJDCTF2020]Easy MD5

 有一个输入框，无论我输入什么都没有回显。于是我尝试扫描目录，可是 buu 有限制。随后我尝试着抓包看看，放到repeater 发现返回包中有提示 select * from 'admin' where password=md5($pass,true) 其实这里的语句应该是有问题的 mysql 里 md5 函数只有一个参数，这应该是 php 内的 md5函数。先测试一下

可以发现在php中当md5的第二个参数值为真时它会出现可利用的字符。
 其实它的第二个参数可以规定函数的输出格式

  • TRUE - 原始 16 字符二进制格式 （这是我们可利用的md5值二进制格式可以当作字符串处理，并不是 01 组成的那个二进制格式）
  • FALSE - 默认。32 字符十六进制数（这是我们一般所见的md5值）

可利用的原始字符串：

  • 129581926211651571912466741651878684928 它对应的md5（*，true）输入是T0Do#'or'8可以达到闭合效果
  • ffifdyop 最常用的 它对应的md5（，true）输出是'or'6]!r,b可以达到闭合效果。

    【二进制数据类型】

这题剩下的md5绕过都可以利用数组绕过

[V&N2020 公开赛]HappyCTFd

  1. 题目进去给了个平台，再根据题目 CTFd 可以猜测这是CTFd平台。然后就可以去百度一下这个平台有什么漏洞了。（当然钱前面尝试了普通方法无法使用admin身份登进去。）
  2. 【CVE-2020-7245 CTFd v2.0.0 – v2.2.2 account takeover分析】 利用方法：
    • 利用添加空格绕过限制来注册一个与受害者用户 名相同的账号。
    • 生成忘记密码链接发送到自己的邮箱
    • 将自己的账号的用户名改成与被攻击者不相同的用户名
    • 用邮箱中收到的链接更改密码即可。
  3. 要用到邮箱但是在内网无法去访问外网，buu提供了内网邮箱。进去注册一个。
  4. 创建一个admin用户，且邮箱为刚刚创建的那个。
  5. 退出登入，再在登入界面选择忘记密码。填入邮箱后会收到一个改密码的链接。
  6. 我们先不去改密码，先用已知的密码去登入我们的账户然后任意改个其它名字。
  7. 点击收到的更改密码链接，此时将要修改的密码是真正的 admmin 用户的。修改密码后以 admin 用户去登入进去，以后发现已经有admin的权限（有新的界面）进去 challenge 下载了附件就是flag

[BJDCTF2020]Mark loves cat

 为什么又是这样，自己想很久想不出来，一看wp就焕然大悟。

  1. 扫描一下题目路径，发现大量的 .git 之类的东西，那么肯定有git泄漏。
  2. 使用 githack 扫描下载源码，发现 一个 flag.php 里面定义了 $flag 为文件读取到的flag。然后index.php 里面有一些php代码。 代码审计了一段时间发现有点绕一直没搞出来。但是后面知道了，不能一直想着最后的echo "the flag is: ".$flag;输出flag。
  3. foreach($_GET as $x => $y){ $$x = $$y; } 这里存在变量覆盖漏洞，也就是我们把flag覆盖赋值给一个变量然后运行exit()退出并把我们的flag抛出来。
  4. 可以利用很多个地方的 exit() 来抛出我们的 flag 。
  • `if($_GET[‘flag’] === $x && $x !== ‘flag’){
    exit($handsome);
    }这里利用的payload是 get一个handsome=flag&flag=handsome第一次循环的时候 if 成立 且此时 flag 已经被我们赋值给了$handsome`
  • if(!isset($_GET['flag']) && !isset($_POST['flag'])){ exit($yds); } 这里我们用的payload是get一个yds=flag虽然本地会因为没设置$_GET[flag]警告但也误伤大雅会出flag。
  • if($_POST['flag'] === 'flag' || $_GET['flag'] === 'flag'){ exit($is); }这里我们使用payload get一个is=flag&flag=flag 虽然 没赋值却调用了$_POST[flag]会有警告但也会出flag。
