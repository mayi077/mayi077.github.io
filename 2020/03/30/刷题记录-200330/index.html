  1. 我们利用\来达到闭合效果。我们传入username=admin\ password=or 1# 最后构成的查询语句类似select * from user where username="admin\" and password="or 1#"这样一来就达到了闭合效果。果然闭合成功，回显BJD needs to be stronger。
  2. 然后我们用脚本来跑出admin的密码。password为^ (ascii(substr((password),%d,1))>%d)#其中^是异或如果猜对了就会返回 true ，所以可以达到效果。贴一个脚本 得到密码后使用admin去登入得到flag。
